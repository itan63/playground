# playground

itan63 の個人用プレイグラウンド。HTML などを試したり、Web ページとして公開したりするための置き場です。

特定のプロジェクト専用ではなく、関係ないファイルでも「とりあえず公開したい／動かして見たい」ものをここに置きます。

## 公開（GitHub Pages）

このリポジトリの HTML は GitHub Pages で Web ページとして公開できます。

- 公開URLの形式: `https://itan63.github.io/playground/<ファイル名>`
- 例: [jfm4.html](https://itan63.github.io/playground/jfm4.html)

### 公開を有効にする手順（初回のみ）

1. リポジトリの **Settings** → **Pages**
2. **Source** を `Deploy from a branch`、**Branch** を `main` / `/(root)` にして **Save**
3. 1〜2分待つと上記URLで表示される

> Pages を使わず手早く確認したいときは htmlpreview でも見られます:
> `https://htmlpreview.github.io/?https://github.com/itan63/playground/blob/main/<ファイル名>`
